# 原價屋 (CoolPC) 估價與採購流程

本技能記錄與原價屋估價網站 (coolpc.com.tw/evaluate.php) 的互動技巧。

## 處理常見障礙
- **彈出視窗**：網站常有促銷彈出視窗或橫幅，遮擋主要操作區域。
    - 方法：先調用 `browser_snapshot` 查看結構，尋找關閉按鈕（通常在右上角的「x」）。若找不到，點擊彈出視窗外或嘗試點擊可能存在的圖片組件進行關閉。
- **網站結構**：估價頁面為大型動態表單，含有大量 `combobox` 與 `iframe`。
    - 方法：使用 `browser_snapshot` 時需注意 `ref` 的有效性，有時需要多次刷新。

## 估價與選購流程
1. **確認新硬體 (如 5070 Ti)**：硬體發售初期可能未直接在個別分類顯示。
    - 技巧：檢查「限量大全配」或「促銷組合」分類，有時新硬體會先出現在組合清單中。
2. **篩選 ITX 組件**：
    - 主機板：在分類中選擇「ITX」作為關鍵字進行篩選。
    - 機殼：確認機殼支援的 CPU 散熱器高度與顯卡長度，尤其是 Mini-ITX 規格。

## 參考步驟
1. 導航至 `https://www.coolpc.com.tw/evaluate.php`。
2. 確認是否被遮擋，如有則執行關閉操作。
3. 根據組裝目標（例如 ITX 規格），依序選擇組件（CPU -> 主機板 -> RAM -> 機殼）。
4. 檢查組合清單確認有無新技術產品（如 RTX 50 系列）。
