# tw-product — 搜尋 MIT 台灣製造品牌

搜尋 [MIT 生活指南](https://mit-products.vercel.app/) 網站，找出符合用戶查詢的台灣製造品牌。

## 觸發條件

當用戶使用 `/tw-product` 指令、或說類似以下語句時觸發：

- 「在 MIT 網站找 XX」
- 「台灣有沒有做 XX 的」
- 「MIT 指南有推薦 XX 嗎」
- 「tw-product XX」

## 網站特性

| 項目 | 說明 |
|------|------|
| 資料來源 | 站長 **版主V**（Drama Note）人工收錄 + 廠商自薦投稿 |
| 品牌總數 | **76 個**（分 6 頁） |
| 分類 | 🍜 美食、👕 服飾配件、👟 各式鞋款、🏠 居家生活、💻 科技電子、✨ 美妝保養、🏔️ 運動、🍼 母嬰用品 |
| 搜尋方式 | **前端 JS 即時過濾**，無 API 端點，必須用 browser 工具 |
| 搜尋行為 | 填入搜尋框即即時過濾，**不需要按 Enter** |
| 詳情頁 | **無**，卡片直接跳外部品牌官網 |

## 工作流程

### Step 1：開啟網站

```bash
browser open https://mit-products.vercel.app
```

### Step 2：取得搜尋框 ref

```bash
browser snapshot -i
# 找到搜尋輸入框，ref 類似 @e38，placeholder 為「搜尋品牌或產品...」
```

### Step 3：填入搜尋關鍵字

```bash
browser fill <search_box_ref> <用戶查詢關鍵字>
browser wait 800
```

**不需要按 Enter**，前端即時過濾。

### Step 4：滾動查看結果

```bash
browser scroll down 1200
browser snapshot -i
```

### Step 5：提取每張品牌卡片

從 snapshot 結果中提取每張 `.card` 的以下資訊：

```
- 品牌名稱：heading level 3 的 text
- 分類：generic 開頭的 text（含 emoji 分類標籤）
- 簡介：generic 的 text 內容（品牌介紹）
- 子分類標籤：text 中類似「保溫水壺」「保溫便當盒」的標籤文字
- 官網連結：link 中有「前往官網」文字的 href
```

### Step 6：格式化回報

```markdown
在 MIT 生活指南找到 **N 個** 符合「關鍵字」的台灣製造品牌：

### 1. 品牌名稱
- **分類**：🏠 居家生活
- **簡介**：立忠實業股份有限公司（三光牌）創立於西元1993年...
- **子分類**：保溫水壺、保溫便當盒、保溫燜燒提鍋
- **官網**：[前往](http://www.leejoin.com.tw/sunkung/)

---
共搜尋 76 個品牌。
```

### Step 7：無結果時

```markdown
MIT 生活指南目前沒有收录「關鍵字」相關的品牌。

目前收錄共 **76 個品牌**，分 8 大類：
- 🍜 美食
- 👕 服飾配件
- 👟 各式鞋款
- 🏠 居家生活
- 💻 科技電子
- ✨ 美妝保養
- 🏔️ 運動
- 🍼 母嬰用品

需要我幫你用其他關鍵字試試看嗎？
```

### Step 8：清理

```bash
browser close
```

## 注意事項

1. **搜尋框 ref 每次都會變**，每次必須先 `snapshot -i` 取得當前 ref
2. **不需要按 Enter**，fill 完等 500-800ms 即可
3. **結果在搜尋框下方**，必須 `scroll down` 才能看到卡片
4. 點「前往官網」會跳到**外部品牌網站**，不是站內詳情頁
5. 全站 76 個品牌，搜尋結果不會超過 76 個

| 餐具、碗盤、瓷器 | `餐具` |

## 進階：多頁搜尋

如果搜尋結果可能跨多頁（例如關鍵字很泛），可以：
1. 點第 2 頁按鈕繼續
2. 重複 Step 5 提取
3. 合併所有頁結果回報

一般情況下 1-2 頁即可覆蓋。
